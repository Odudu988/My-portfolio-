3. Quiz App

A dynamic quiz app which presents students with multiple choice questions to pick from and a timer and at the end it calculates their scores according to a ccoeffficient system.

Requirements

The application has the following features:

  • The user will be presented with a "start" button and some details about the quiz.
  • When the user presses start they're presented with the first multiple choice question.
  • The questions are to be presented as cards with the answer being buttons on the card.
  • When the user selects an answer, the button are to turn red or green depending on the result.It should also show what the correct anwer was.
  • If the user answers correctly a score of 1 is to be added and if the user does'nt attempt the question the score is reduced by 1.
  • There is a timer of 1-2 minutes for each question, if the user does'nt answer in that time frame it would skip to the next quesion and reduce their score by 1.
  • At the end of the quiz, the user is presented with a final score which is their total mark multiplied by the set coefficient that the admin had already set.

Technologies used:

  • HTML
  • CSS
  • Javascript
